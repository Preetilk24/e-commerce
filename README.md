# E-Commerce Platform

A full-stack e-commerce application built with MERN stack (MongoDB, Express.js, React.js, Node.js).

## Tech Stack

### Frontend
- React.js
- Redux for state management
- CSS for styling
- Axios for API requests

### Backend
- Node.js
- Express.js
- MongoDB
- JWT for authentication

## Getting Started

### Prerequisites
- Node.js
- MongoDB
- npm or yarn
